### Swagger sample

### Installation

`npm install`

### Setup DB
`npm run setup:db`

### Running the api
`npm run start:dev`

### Running

Once the application is running you can visit [http://localhost:9999/api](http://localhost:3000/api) to see the Swagger interface.

See [here](https://docs.nestjs.com/recipes/swagger#bootstrap) for more information.
